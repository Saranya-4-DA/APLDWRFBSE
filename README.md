Acoustic Pipeline Leak Detection and Leak Size Estimation

This project presents an intelligent pipeline monitoring system that detects water leaks and estimates leak sizes using vibration signal analysis. The system combines Deep Learning and Machine Learning techniques to provide real-time leak detection through an interactive Streamlit dashboard.

The solution utilizes a 1D Convolutional Neural Network (1D-CNN) for feature extraction and leak classification, followed by a Random Forest Regressor for leak size estimation. The dashboard visualizes vibration signals, leak status, and predicted leak sizes in real time.
